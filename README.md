# Keepsake

A personal web application for tracking and exploring sports-related collections and experiences through an interactive interface.

## Features

- **Interactive 3D Globe** - Visualise your collection geographically
- **Collection Tracking** - Log badges, scarves, shirts, and more
- **Stadium Visits** - Track stadiums you've visited and matches you've attended
- **Multiple Views** - Switch between globe, list, and grid views
- **Search & Filter** - Find items by club, country, type, or date
